# My_Chat
A chatbot made using python library nltk 

To use this program:

<strong> <h1> Run the following commands in cmd </h1> </strong>

To download the files:
  ```
  git clone https://github.com/Hat1412/My_Chat.git
  cd My_Chat
  ```
To run file:
  ```
  #In windows:
  python main.py
  #In Linux/MacOS
  python3 main.py
  ```
  
 Debug any errors:
  ```
  pip install nltk
  python
  import nltk
  nltk.download("stopwords")
  ```
